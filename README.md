# materialtheme
Application that uses good practices with the use of material design.

# Install
- Android Studio 2.3 or Higher
- Build Tools 23+
- Target SDK: API Level 26
- git clone https://github.com/rodrigogaraujo/culinaria.git

# Libraries & Features
- RecyclerView + GridView
